<task type="auto" tdd="false">
  <name>Task 3: Audit atoms for stateless compliance</name>
  <files>src/common/components/atoms/</files>

    - src/common/components/atoms/SeverityBadge.tsx

  <action>
Check each atom for stateless compliance:

1. **No useState/useEffect** - atoms should not manage data state
2. **Props interface** - all data comes from props
3. **UI state allowed** - open/close, hover states are acceptable if contained

Check for:

- useState calls (except for UI toggle)
- useEffect with data fetching
- Context access for data (UI context like theme is OK)

For any violations:

- Move state management to parent component
- Pass data via props
- Document if state is UI-only
  </action>
  <acceptance_criteria>
- No useState for data management in atoms
- No useEffect for data fetching in atoms
- All data passed via props
- UI state (hover, focus) is acceptable
  </acceptance_criteria>
  <verify>
  <automated>grep -E "useState|useEffect" src/common/components/atoms/\*.tsx | grep -v "// UI state" && echo "STATE_FOUND" || echo "OK"</automated>
  </verify>
  <done>Atoms audited for stateless compliance</done>
