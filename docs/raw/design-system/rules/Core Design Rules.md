# Astra — Core Design Rules

> **Canonical source:** `design.md §1 — Brand Philosophy` is the single source of truth for these rules. This document provides the verbose format for deeper reading. If the two ever conflict, design.md §1 takes precedence.

*(Premium Product Design Principles)*

## Rule 1 — Radical Simplicity

Astra interfaces must remove anything that does not directly support clarity or function.

Every element should have a clear purpose.

If an element does not improve usability, hierarchy, or comprehension, it should be removed.

Design should feel:

* calm
* minimal
* intentional

Avoid:

* decorative UI
* visual noise
* unnecessary UI elements

**Principle:**
Clarity is the ultimate form of sophistication.

---

# Rule 2 — Precision in Spacing

Spacing must follow strict alignment and rhythm.

Layouts should feel engineered rather than assembled.

Spacing should always:

* follow consistent increments
* maintain balanced margins
* align to a grid structure

Visual rhythm should be predictable and harmonious.

Avoid:

* inconsistent spacing
* elements placed without alignment
* uneven margins

**Principle:**
Precision creates trust.

---

# Rule 3 — Typography Leads the Interface

Typography should carry the majority of the visual hierarchy.

Design should rely more on:

* typography
* spacing
* contrast

rather than decoration.

Headings should feel confident and readable.

Body text should feel effortless to scan.

Avoid:

* excessive font weights
* overly stylized type
* dense text blocks

**Principle:**
Great typography eliminates the need for visual noise.

---

# Rule 4 — Color is a Guidance Tool

Color should be used intentionally to guide attention.

Most of the interface should remain neutral.

Accent colors should highlight:

* primary actions
* important states
* key interactions

Avoid:

* overusing accent colors
* decorative color usage
* inconsistent color meaning

**Principle:**
Color directs focus, not decoration.

---

# Rule 5 — White Space is a Feature

White space should be treated as an essential design component.

Layouts should feel spacious and breathable.

Spacing should help users:

* understand hierarchy
* scan content quickly
* maintain focus

Avoid:

* dense layouts
* cramped interfaces
* crowded components

**Principle:**
Space improves comprehension.

---

# Rule 6 — Depth Must Be Subtle

Visual depth should be created through subtle cues such as:

* soft shadows
* layering
* contrast

Depth should never feel exaggerated.

Avoid:

* heavy drop shadows
* aggressive gradients
* glossy UI effects

**Principle:**
Depth should enhance clarity, not draw attention.

---

# Rule 7 — Motion Must Be Purposeful

Animation should serve functional goals such as:

* indicating state change
* guiding user attention
* reinforcing interaction feedback

Motion should feel:

* smooth
* quick
* unobtrusive

Avoid:

* decorative animations
* slow transitions
* distracting effects

**Principle:**
Motion should explain interaction.

---

# Rule 8 — Consistency Builds Trust

Every interface element should behave predictably.

Consistency must be maintained in:

* spacing
* typography
* color usage
* interaction patterns

Inconsistent design creates cognitive friction.

**Principle:**
Consistency reduces effort.

---

# Rule 9 — Detail Reflects Craftsmanship

Premium products are defined by attention to detail.

Designers should carefully consider:

* alignment
* micro spacing
* text balance
* component proportions

Small inconsistencies can reduce perceived quality.

**Principle:**
Craftsmanship is visible in details.

---

# Rule 10 — Restraint Defines Premium Design

The most premium designs are defined by what they **choose not to include**.

Design should prioritize:

* clarity
* balance
* refinement

Avoid trends that sacrifice longevity for novelty.

**Principle:**
Restraint creates timeless design.

---

# Astra Design Philosophy (Summary)

The Astra visual identity is defined by three core qualities:

**Precision**
Design feels engineered and intentional.

**Clarity**
Information is immediately understandable.

**Restraint**
Minimalism is used to elevate focus and elegance.
