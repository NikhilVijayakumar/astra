# 🧭 BAVANS – Developer & Design Directive  
*Non-Negotiable Experience Guidelines*

This document defines the **experience philosophy** for the BAVANS website.  
All design and development decisions must align with these principles.

---

## 1. Core Theme

**Modern Minimalist Tech-Noir**

- Dark-first interface (Black / Deep Navy base)
- High-contrast typography
- Selective accent usage (Gold / Neon tones)
- Premium, cinematic feel — never playful or cartoonish
- The interface should feel like a **living archive**, not a marketing page

> If an element does not add meaning, remove it.

---

## 2. Interaction Philosophy

**Micro-Interactions Are Mandatory (2026 Standard)**

Every user action must receive subtle, intentional feedback:

- Hover → slight motion or glow
- Click → tactile response
- Scroll → progressive reveal
- Focus → clarity, not distraction

Interactions should feel:
- Calm
- Purposeful
- Human-paced

No aggressive animations. No gimmicks.

---

## 3. Motion & Animation Rules

- Motion must **explain**, not decorate
- Prefer:
  - Fade + translate
  - Scale with restraint
  - Scroll-based storytelling
- Avoid:
  - Infinite loops
  - Flashy transitions
  - Excessive easing

> If motion draws attention to itself, it’s wrong.

---

## 4. Typography as a First-Class Citizen

- Headlines carry meaning
- Oversized typography is encouraged
- Copy must breathe (generous spacing)
- Text hierarchy must be obvious without color

Typography is part of the storytelling system.

---

## 5. Performance Expectations

- Fast initial load
- Motion must not block interaction
- Graceful degradation on low-end devices
- No heavy assets unless justified

Performance is a feature, not an optimization.

---

## 6. Accessibility & Trust

- Readable contrast at all times
- Keyboard navigability
- Motion respects reduced-motion preferences
- Clear, honest language — no hype

Trust is part of the brand.

---

## 7. Explicit Non-Goals

- No trend-driven UI
- No excessive gradients
- No auto-playing audio
- No unnecessary 3D/WebGL

---

## 8. Decision Rule

When unsure, ask:

> “Does this make the idea clearer, calmer, and more timeless?”

If not — remove it.

---

## 9 Theme Modes

BAVANS supports both **Dark** and **Light** themes.

- **Dark Mode:** Canonical experience  
  - Deep Navy / Black base  
  - High-contrast typography  
  - Gold / Neon accents  

- **Light Mode:** Alternative viewing mode  
  - Warm off-white surfaces  
  - Charcoal typography  
  - Identical layout, motion, and hierarchy  

Theme switching must never alter:
- Typography scale
- Spacing system
- Interaction behavior
- Narrative flow

Tech-Noir is preserved through structure, contrast, and motion — not color alone.

---

*This directive applies across all future BAVANS digital properties unless explicitly revised.*
