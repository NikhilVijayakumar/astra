# 🎨 BAVANS – Design Reference Matrix  
*Visual Language, Interaction Style & Storytelling References*

This document maps **BAVANS website sections** to **best-in-class design references**.  
The goal is **inspiration and alignment**, not duplication.

---

## 🧭 Global Design Philosophy

- Story-first, not feature-first
- Minimalist, premium, timeless
- Motion reveals meaning (never decorative)
- Performance > spectacle
- Vision without exposing internals

## 🎯 How to Use This Document

- These references define **mood, pacing, and interaction quality**
- They are **not UI kits** and should not be copied pixel-for-pixel
- Antigravity may:
  - Suggest alternatives
  - Propose hybrid approaches
  - Replace references with equivalent-quality work

---

## 🚫 Explicit Non-Goals

- No heavy WebGL showcases
- No chatbot UI metaphors
- No loud gradients or trend-chasing
- No over-animated pages

---

## ✅ Success Criteria

A successful BAVANS website should feel:
- Calm but powerful
- Visionary but grounded
- Creative yet trustworthy
- Designed, not decorated
